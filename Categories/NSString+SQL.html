– stringWithSQLParams:

put params in string. The key in string should start with colon, and the key in Params don’t. this method will add colon before matching.

- (NSString *)stringWithSQLParams:(NSDictionary *)params

Parameters

params

params in string to replace

Return Value

return a string.

Discussion

put params in string. The key in string should start with colon, and the key in Params don’t. this method will add colon before matching.

Example:

NSString foo = @“hello :bar”; // the key in string must start wich colon NSDictionary params = @{ @“bar”:@“world” // don’t put colon in key };

will generate @“hello world”
